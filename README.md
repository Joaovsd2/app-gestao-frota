# 🚗 Sistema de Gestão de Frota

Aplicação desenvolvida para gerenciamento de veículos, com foco em controle de consumo, organização de dados e validação de informações.

---

## 📌 Funcionalidades

* Controle de abastecimentos, viagens e manutenções
* Registro de despesas operacionais
* Cálculo automático de consumo (km/l)
* Histórico de utilização dos veículos
* Geração de dados para análise

---

## 🔌 Integrações e APIs

* Integração com serviços de backend para persistência de dados
* Consumo de APIs REST para operações de leitura e escrita
* Tratamento de respostas e validação de dados recebidos
* Comunicação entre aplicação e serviços externos

---

## 🧪 Testes e Validação

* Testes funcionais nas principais funcionalidades
* Validação de entradas para evitar inconsistências
* Verificação de integridade dos dados armazenados
* Testes de comunicação com serviços externos

---

## 🛠️ Tecnologias Utilizadas

* Flutter
* Dart
* Firebase
* Banco de dados NoSQL
